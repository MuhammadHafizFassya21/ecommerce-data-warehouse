# LinkedIn Post Draft

I recently built an end-to-end Data Engineering project: E-Commerce Data Warehouse & Analytics Dashboard.

This project processes raw e-commerce data into structured analytical layers using Python, PostgreSQL, SQL, and Flask.

Project highlights:

- Raw data ingestion from CSV to PostgreSQL
- Staging layer for data cleaning and standardization
- Warehouse layer using fact and dimension tables
- Data mart layer for business-ready reporting
- SQL business analysis queries
- Custom web analytics dashboard using Flask and Chart.js
- Year and month filter for interactive analysis

The dashboard helps analyze sales performance, product category performance, seller delivery performance, customer location performance, payment behavior, and the relationship between late delivery and customer review score.

Tech stack:

Python, PostgreSQL, SQL, Flask, SQLAlchemy, HTML, CSS, JavaScript, Chart.js, DBeaver, GitHub.

This project helped me practice data ingestion, data modeling, SQL transformation, data mart design, API development, and dashboard development in one complete workflow.
