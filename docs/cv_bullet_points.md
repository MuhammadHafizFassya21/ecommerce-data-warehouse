# CV Bullet Points

## English Version

- Built an end-to-end e-commerce data warehouse using Python, PostgreSQL, and SQL to process raw order, customer, seller, product, payment, delivery, and review data.
- Designed raw, staging, warehouse, and data mart layers to support structured data transformation and business reporting.
- Created fact and dimension tables using a star schema approach for order-level and item-level analysis.
- Developed business-ready data marts for monthly sales, product category performance, seller delivery performance, customer location analysis, payment behavior, and review analysis.
- Built a custom Flask-based analytics dashboard with PostgreSQL integration, REST API endpoints, Chart.js visualizations, and year/month filters.

## Indonesian Version

- Membangun end-to-end data warehouse e-commerce menggunakan Python, PostgreSQL, dan SQL untuk mengolah data order, customer, seller, produk, pembayaran, pengiriman, dan review.
- Merancang layer raw, staging, warehouse, dan data mart untuk mendukung proses transformasi data dan pelaporan bisnis.
- Membuat tabel fact dan dimension menggunakan pendekatan star schema untuk analisis order dan item transaksi.
- Membangun data mart untuk analisis penjualan bulanan, performa kategori produk, performa pengiriman seller, wilayah customer, metode pembayaran, dan ulasan pelanggan.
- Mengembangkan custom analytics dashboard berbasis Flask yang terhubung ke PostgreSQL, dilengkapi API endpoint, visualisasi Chart.js, serta filter tahun dan bulan.
